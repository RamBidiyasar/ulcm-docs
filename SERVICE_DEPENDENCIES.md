### 5. DLR Pipeline

> End-to-end Delivery Report flow — from channel response to enriched event in Kafka.

```mermaid
sequenceDiagram
    participant CH as External Channel<br/>(SMS / Email / Push / RCS / WhatsApp)
    participant DLRAPI as uclm-dlr-api-service
    participant KAFKA as Apache Kafka<br/>(DLR topics)
    participant DACL as uclm-dlr-aerospike-cache-loader
    participant AS as Aerospike
    participant DLRE as uclm-dlr-enricher
    participant KAFKA2 as Apache Kafka<br/>(enriched DLR topics)

    CH->>DLRAPI: Delivery Report (HTTP callback)
    DLRAPI->>KAFKA: Publish raw DLR event<br/>(channel-specific topic)

    par Cache Loader
        KAFKA-->>DACL: Consume DLR event
        DACL->>AS: Write/update DLR cache entry
    and Enricher
        KAFKA-->>DLRE: Consume DLR event
        DLRE->>AS: Lookup campaign/user metadata
        AS-->>DLRE: Metadata response
        DLRE->>KAFKA2: Publish enriched DLR event
    end
```
